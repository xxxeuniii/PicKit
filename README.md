# PicKit - 图片工具箱

一个简单高效的在线图片处理工具。

## 在线访问

https://pic-kit.vercel.app/

## 功能

- **图片压缩**：调整质量和尺寸，减小文件体积
- **图片裁剪**：自由裁剪，支持多种比例
- **格式转换**：PNG、JPEG、WebP 格式互转
- **批量重命名**：批量重命名图片并打包下载
- **批量裁剪**：批量处理多张图片
- **图片转PDF**：将图片合并成PDF文件

## 技术栈

- Vue 3 + Vite
- Element Plus
- Canvas API
- Pica (图片缩放)
- Cropper.js (图片裁剪)

## 使用

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建生产版本
npm run build
```

## 特性

- 纯前端实现，数据安全
- 无需注册登录
- 支持中文、英文、日文
- SEO 优化：已配置 meta 信息、sitemap 和 robots.txt
